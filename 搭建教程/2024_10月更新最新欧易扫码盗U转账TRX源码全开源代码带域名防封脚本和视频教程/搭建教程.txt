扫码支付TRX授权 搭建教程 配合域名防封脚本使用 

原理：通过github提供的免费域名进行防封操作，因为github的权重比较高也不会马上封禁，并且我们可以生成非常多的github域名，每次只需要更换用户名就可以得到新的域名。

逻辑：设置一个主域名，客户扫码进入主域名，由主域名将客户带到github的域名中。（具体可以看python代码，全开源）

缺点：每次只能允许一个客户同时访问，因为进入第二次之后域名就会被封了！

我们只需要准备一个 主域名就行了  将域名设置为二维码 （比如使用草料二维码） 再将二维码PS到钱包的收款码发给客户扫

创建一个目录存放脚本  服务器需要安装 Nginx 
建议使用 Ubuntu 系统 自带python 

进入终端  

1.切换到root用户（如果已是root权限请忽略）
sudo -i

2.安装虚拟环境模块（如果已经安装请忽略）
sudo apt install python3-venv

3.进入脚本目录
cd 目录
例如：cd /www/wwwroot/guthub

3.创建虚拟环境目录 名称为hj（如果已经创建请忽略）
python3 -m venv hj

4.进入虚拟环境目录 名称为hj
source hj/bin/activate

5.安装谷歌浏览器  安装好了 之后继续安装漪蓝裤
wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
sudo apt install ./google-chrome-stable_current_amd64.deb

6.安装脚本的依赖库
pip install python-dotenv selenium webdriver-manager flask flask-cors

7.启动一个会话
tmux new -s myscript

8.启动脚本
python github.py  在启动之前先设置我们的  登录账号密码和仓库名字

重连会话命令：tmux attach -t myscript
关闭程序命令：CTRL+C

9.给脚本设置一个主域名：添加站点-申请SSL-反向代理

脚本启动之前请先设置好你的github账号密码
因为github不提供 修改用户名的api  所以只能是 模拟点击
